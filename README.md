# 獅尾圓體 Swei Gothic

獅尾圓體基於[思源黑體](https://github.com/adobe-fonts/source-han-sans)的拔腳和加圓改造，更加簡明現代化的字體。支援简体中文、繁體中文、韓文與日文。

「獅尾圓體」、「源泉圓體」、「jf open 粉圓」的比較：

![字體比較](https://github.com/max32002/swei-gothic/raw/master/preview/compare_swei_gothic.png)

* 在「刂」的筆畫，粉圓和獅尾比較相似台灣教育部推薦字體筆順。
* 在「肉」的筆畫，獅尾比較相似台灣教育部推薦字體筆順。
* 在「糸」的筆畫，獅尾比較相似台灣教育部推薦字體筆順。
* 在「女」的筆畫，獅尾比較相似台灣教育部推薦字體筆順。
* 在「辶」的筆畫，獅尾比較相似台灣教育部推薦字體筆順。
* 在「食」的筆畫，獅尾比較相似台灣教育部推薦字體筆順。
* 在「草」部的筆畫，獅尾是分開的二個部份。

## 附註

* 這不是一個專業的字型檔案，大約每100個字，會有3個字怪怪，或是可以看到思源黑體的的直角。
* 希望有贊助者，或其他勇者繼續改良和調整很多怪怪的小細節。
* 由於小編對字體編碼方式完全不清楚，相較於原版的思源黑體，可能有掉一些符號或不常用的字。
* 專業的網友 But Ko 幫忙指出問題：OS/2表是掛掉的，另外GSUB、GPOS、vmtx表都不見了，那應該所有直排、異體字選擇、kerning資訊都丟失了。（例如直排標點旋轉）。在Adobe軟體裡可能會碰到無法預期的錯誤。
